> Esse repositório não reflete a minha configuração atual do neovim. Eu decidi
> mantê-lo apenas para preservar o histórico de mais de dois anos de
> modificações que eu tenho aqui, para o caso de ser interessante para alguém
> (ou para mim mesmo no futuro). Minha configuração atual encontra-se no meu
> repositório de [dotfiles](https://github.com/eduardo-antunes/dotfiles).

Essa é a minha configuração pessoal do [neovim](https://neovim.io/), um editor
de texto livre de terminal. Ela requer o neovim versão 0.12, no mínimo, e
funciona melhor quando são instalados os programas `ripgrep` e `fd` (alguns
plugins os utilizam automaticamente para obter um melhor desempenho e/ou
usabilidade).
